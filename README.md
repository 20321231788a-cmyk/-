# 二手车直播投流 SaaS 助手

这是一个面向二手车门店的抖音直播投流管理网页原型，覆盖账号数据看板、视频分析、直播复盘、线索跟进、投放计划草稿、直播运营助手、Codex AI 工作台和智能体参与计划。

## 核心功能

- 经营看板：消耗、线索、预约、到店、成交、CPL、成交成本。
- 视频分析：根据播放、互动、涨粉、主页访问、线索和花费判断是否适合 DOU+ 加热。
- 直播场次：复盘直播观看、私信、表单、预约、到店、成交和消耗。
- 线索跟进：展示线索来源、预算、需求、状态、销售负责人和优先级。
- 投放计划：生成 DOU+、巨量本地推、线索广告计划草稿，人工确认后再去官方平台执行。
- 直播助手：评论分类、自动回复建议、讲车脚本和销售承接话术。
- AI 工作台：导出账号快照给 Codex，导入后可转成下一轮投流计划。
- 智能体参与计划：分为用户智能体、策划智能体、执行智能体、验收智能体。
- 数据导入：支持表格快速导入和 AI/Codex 数据包导入两种模式。
- AI 编程工具连接：提供 CLI 桥接脚本，并预留 MCP 资源和工具设计。

## 智能体分工

### 用户智能体

- 输入：用户身份、当前页面、数据缺口、待办任务、最近一次操作。
- 输出：新手引导、缺数据提醒、下一步建议、风险提示、可复制指令。
- 验证：不替用户做高风险确认，优先给最短可执行路径。

### 策划智能体

- 输入：账号快照、车源结构、预算、历史投放、线索质量。
- 输出：周投流策略、内容选题、预算拆分、人群与地域建议。
- 验证：预算不能超过门店上限，策略必须包含加投和停投规则。

### 执行智能体

- 输入：策划计划、视频素材、直播排期、车辆库存。
- 输出：DOU+ 草稿、本地推草稿、直播脚本、评论回复、销售跟进清单。
- 验证：不自动发布广告，敏感承诺人工确认，回复只做建议。

### 验收智能体

- 输入：消耗、线索、有效线索、预约、到店、成交、直播数据。
- 输出：日复盘、素材评分、投放验收、下轮优化建议。
- 验证：以有效线索、到店和成交为主，标记异常数据和归因缺口。

## 运行

```powershell
npm install
npm run dev
```

构建：

```powershell
npm run build
```

## CLI / MCP 连接

生成给 Codex 或其他 AI 编程工具读取的数据包：

```powershell
npm run bridge -- sample
```

打印可复制提示词：

```powershell
npm run bridge -- prompt
```

校验 AI 生成的策略结果：

```powershell
npm run bridge -- validate data/inbox/strategy_result.json
```

更多 MCP 资源和工具规划见 [MCP_CONNECT.md](./MCP_CONNECT.md)。

## 边界

第一版是前端 SaaS 原型，使用样例数据和 CSV 导入入口。官方巨量引擎、DOU+、本地推接口以配置和流程预留为主，不自动创建或修改真实广告计划。
